Comment récupérer une clé NVIDIA NIM (gratuit, environ 5 min)

Vérification téléphone requise. NVIDIA exige un numéro de téléphone. Serveurs parfois saturés. Si la vérification SMS échoue, réessayer plus tard.

  1. Aller sur build.nvidia.com, saisir l'email.
  2. Confirmer l'email (code reçu), puis vérifier le numéro (SMS avec code).
  3. Créer le compte NVIDIA Cloud, nom au choix (par exemple perso).
  4. Ouvrir build.nvidia.com/settings/api-keys, cliquer Generate API Key.
  5. Donner un nom (par exemple penates), copier la clé nvapi-…, elle ne se ré-affiche pas.
  6. Coller la clé ici, cliquer Tester puis Enregistrer.

Clé stockée uniquement dans le navigateur (localStorage). Pénates ne la voit pas, sauf le temps d'un OCR (transmise à NIM, jamais stockée serveur). Code source consultable.

Bienvenue dans Pénates

Préserve la connaissance des prix.

Outil minimaliste pour mémoriser les prix HDV de Dofus Retro entre les sessions. Les données restent dans le navigateur, personne d'autre n'y accède.

Pourquoi Pénates ? (le projet et le nom)

Le projet. Dofus Retro 1.29 n'affiche pas de prix moyen sur l'HDV, contrairement à d'autres versions. Évaluer une vente juste oblige donc à comparer, retenir, ou griffonner. Pénates remplace ce griffonnage : un bloc-note avec catalogue intégré, calcul de coût de craft, OCR, dossiers de farm, marges.

Le nom. Dans la mythologie romaine, les Pénates sont les divinités protectrices du foyer, des provisions et des biens domestiques (« regagner ses pénates »). Garder chez soi la connaissance des prix, comme on garderait ses provisions.

Le clin d'œil Dofus. Les Pénates du Corbac, en 1.29, abritent la Bibliothèque du Maître Corbac (donjon niv 110). Le lore dit qu'y siégeait jadis la Confrérie des Pénates, ordre voué à la préservation du savoir. Le boss Maître Corbac renvoie à Maître Corbeau de La Fontaine, qui perd son fromage faute d'en connaître la valeur. Mémoriser ses prix, c'est ne pas finir comme lui.

Projet open source, sans collecte de données, sans publicité.

L'OCR, qu'est-ce que c'est ?

L'OCR (Optical Character Recognition) est une technologie qui lit automatiquement le texte d'une image. Dans Pénates : une capture d'écran d'une fiche d'item HDV est envoyée à un modèle d'IA de vision qui identifie nom, niveau, prix x1/x10/x100, puis remplit ces champs dans l'app sans saisie manuelle.

L'IA n'est utilisée que pour ça. Pas de génération de texte, pas d'analyse des données utilisateur, pas de suggestion. Sans clé NIM, l'app fonctionne pareil, les prix sont saisis à la main.

Quoi capturer, et comment (sans rien installer)

Capturer la fiche d'un item à l'HDV, avec les 3 colonnes de prix (x1, x10, x100) lisibles. Pas besoin de tout l'écran.

  • Windows 10/11 : Win+Shift+S, sélectionner, puis Ctrl+V sur la zone de drop.
  • macOS : Cmd+Shift+4, sélectionner, puis glisser le fichier généré.
  • Linux : Print ou gnome-screenshot, KSnip, même flow.

Modes acceptés : drag-drop, coller (Ctrl+V), fichier. Pas besoin d'un logiciel tiers (ShareX, etc.).

Activer l'OCR : récupérer une clé NVIDIA NIM (gratuit, ~5 min)

Téléphone requis. NVIDIA exige un numéro pour activer l'API. Serveurs parfois saturés : si le SMS échoue, réessayer plus tard.

  1. Aller sur build.nvidia.com, saisir un email.
  2. Confirmer l'email (code reçu) + vérifier le numéro (SMS).
  3. Créer le compte NVIDIA Cloud (nom au choix, ex. perso).
  4. Ouvrir /settings/api-keys puis Generate API Key.
  5. Nom au choix (ex. penates), copier la clé nvapi-… (elle ne se ré-affiche pas).
  6. Revenir ici, cliquer J'ai une clé, coller la clé.

Quota ~1000 OCR/mois, aucune CB. Clé stockée en localStorage, jamais sur le serveur Pénates. Code source consultable.
