Comment récupérer une clé NVIDIA NIM (gratuit, environ 5 min)

Vérification téléphone requise. NVIDIA exige un numéro de téléphone. Serveurs parfois saturés. Si la vérification SMS échoue, réessayer plus tard.

  1. Aller sur build.nvidia.com, saisir l'email.
  2. Confirmer l'email (code reçu), puis vérifier le numéro (SMS avec code).
  3. Créer le compte NVIDIA Cloud, nom au choix (par exemple perso).
  4. Ouvrir build.nvidia.com/settings/api-keys, cliquer Generate API Key.
  5. Donner un nom (par exemple penates), copier la clé nvapi-…, elle ne se ré-affiche pas.
  6. Coller la clé ici, cliquer Tester puis Enregistrer.

Clé stockée uniquement dans le navigateur (localStorage). Pénates ne la voit pas, sauf le temps d'un OCR (transmise à NIM, jamais stockée serveur). Code source consultable.

Bienvenue dans Pénates

Préserve la connaissance des prix.

Outil minimaliste pour mémoriser les prix HDV de Dofus Retro entre les sessions. Les données restent dans le navigateur, personne d'autre n'y accède.

Pourquoi Pénates ? (l'histoire derrière le projet et le nom)

Le projet est né d'un constat simple : en revenant sur Dofus Retro, l'HDV n'affiche pas de prix moyen sur les items, ce qui rend l'évaluation pénible et oblige à tout retenir de tête. Pénates est juste un bloc-note dédié, en mieux : catalogue intégré, calcul de craft, OCR, dossiers de farm, suivi de marges.

Le nom vient de la mythologie romaine. Les Pénates sont les divinités protectrices du foyer, des provisions, et des biens domestiques. Le terme désigne aussi, par extension, ce qui appartient à un foyer (« regagner ses pénates »). Métaphore directe : garder chez soi la connaissance des prix.

Le clin d'œil Dofus. Les Pénates du Corbac sont une zone du jeu en 1.29, qui abrite la Bibliothèque du Maître Corbac (donjon niveau 110). Selon le lore, ce lieu hébergeait jadis la Confrérie des Pénates, ordre dédié à la préservation de la connaissance. Et un peu plus loin, Maître Corbac rappelle évidemment Maître Corbeau de La Fontaine, qui perd son fromage faute d'en connaître la valeur. Mémoriser ses prix, c'est ne pas finir comme lui.

Projet open source, sans collecte de données, sans publicité. Si quelqu'un trouve ça utile, tant mieux. Si ça inspire un meilleur outil, encore mieux.

L'OCR, qu'est-ce que c'est ?

L'OCR (Optical Character Recognition) est une technologie qui extrait automatiquement le texte présent sur une image.

Dans le cas de Pénates, l'app reçoit une capture d'écran d'une fiche d'item à l'HDV. Un modèle d'IA de vision identifie le nom, le niveau, et les prix x1/x10/x100 affichés, puis remplit automatiquement ces champs dans l'app sans saisie manuelle.

L'IA est utilisée uniquement pour cette lecture des captures HDV. Pas de génération de texte, pas d'analyse des données utilisateur, pas de suggestion automatique. Sans clé NIM, l'app fonctionne pareil, les prix sont simplement saisis à la main.

Quoi capturer, et comment (sans rien installer)

Ce qu'il faut capturer : la fenêtre d'un item dans l'HDV de Dofus, avec les 3 colonnes de prix (x1, x10, x100) lisibles. Pas besoin de tout l'écran, seulement la fiche.

Comment capturer, en utilisant les outils intégrés au système d'exploitation :

  • Windows 10/11 : Win+Shift+S, sélectionner la zone, puis Ctrl+V sur la zone de drop.
  • macOS : Cmd+Shift+4, sélectionner la zone, puis glisser le fichier généré sur la zone de drop.
  • Linux : Print Screen ou outil intégré (gnome-screenshot, KSnip), même flow.

Pénates accepte trois modes d'entrée : drag-drop d'image, coller (Ctrl+V), sélection de fichier. Pas besoin de logiciel tiers (ShareX, etc.).

Activer l'OCR : récupérer une clé NVIDIA NIM (gratuit, environ 5 min)

Vérification téléphone requise. NVIDIA exige un numéro de téléphone pour activer l'API. Les serveurs NVIDIA sont parfois saturés. Si la vérification SMS échoue, réessayer plus tard.

  1. Aller sur build.nvidia.com, saisir un email pour se connecter ou créer un compte.
  2. Confirmer l'email (code reçu par mail), puis vérifier le numéro de téléphone (SMS avec code).
  3. Créer un compte NVIDIA Cloud. Le nom du compte n'a pas d'importance (par exemple perso).
  4. Ouvrir build.nvidia.com/settings/api-keys puis cliquer Generate API Key.
  5. Donner un nom à la clé (par exemple penates), valider. Copier la clé nvapi-… affichée, elle ne se ré-affiche pas.
  6. Revenir ici, cliquer J'ai une clé ci-dessous, coller la clé dans Réglages.

Quota gratuit environ 1000 OCR par mois, aucune carte de crédit demandée. Sécurité : la clé est stockée uniquement dans le navigateur (localStorage), Pénates ne la voit pas, sauf le temps d'un OCR pour la transmettre à NIM (jamais stockée serveur). Code source consultable sur GitHub.
